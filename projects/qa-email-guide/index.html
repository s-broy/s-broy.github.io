In Umbraco, disclaimer codes appear superscripted and within brackets; for example: [LEGALTERM].

The disclaimer code is a disclaimer that was created in Umbraco with a specific name, usually labelled something related to what they are disclaiming.

Umbraco automatically replaces the disclaimer code in brackets with numbers in the preview link and in emails. Thus, Umbraco automatically numbers them based on the order they appear in. A disclaimer code that appears more than once in the email gets the same number as when it first appeared in the email.

This is a module image.

In Umbraco, most modules can get assigned an image and a CTA. An image has a link, an alias and an ALT text field. A CTA has an alias and a link field.

Single quotes use the straight apostrophe HTML code. Double quotes use the straight quotation mark code. None of these should ever be curly or smart. If they are, recommend writing them out using their respective HTML code or entity.

Trademark, registered, and copyright signs never go in <sup>, because the ToyotaType font automatically superscripts them. This proprietary font isn't supported within email clients, only in Web view, so these signs appear larger within email clients.

These signs also never go in <strong> tags, because they make them difficult to read, especially on mobile devices.

key line

This is the first line of the Toyota footer and is usually bold, but it actually depends on what Toyota footer is being used.

There are four kinds of footers: (1) commercial; (2) transactional; (3) dealer; and (4) vendor.

The difference between the commercial and transactional footers are that the commercial footer has two unsubscribe links. One is in one of the paragraphs and the second is in the list of links below.

Transactional footers don't have the two unsubscribe links. The content is also slightly different and is a bit shorter than the commercial footer.

Dealer footers have a lot of content, as the dealer name appears twice, once in a paragraph and another at the very bottom of the email after "This email was sent by." It also has two unsubscribe links like the commercial footer. However, in the dealer footer, one is to unsubscribe from Toyota emails and the other is to unsubscribe from dealer emails. It doesn't have Contact Us and Privacy Rights links. The last line is the copyright line, but it is much shorter.

Lastly, as long as the vendor footer is displayed in the emails it is supposed to, that would satisfy QA's validation. Because it is inserted and provided by the vendor, comparing it to the source, which is usually located in the vendor footer sheet in the URL matrix, suffices. Any issues may be logged, but it is more of a courtesy so the PM can inform the vendor, because we're not able to make any vendor edits within our organization.
